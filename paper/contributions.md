# Contributions

1. A modular Python research package for deterministic studies of uncertainty-aware navigation in unknown grid environments.
2. A risk-aware planning prototype that combines occupancy probability, tail-risk estimation, and returnability penalties.
3. A runtime monitoring and safe-mode supervision layer that separates safety checks from planner internals.
4. A reproducible benchmarking workflow with YAML configuration, deterministic seeds, CSV outputs, and Markdown summaries.
5. Automated generation scripts for architecture diagrams, risk heatmaps, trajectory visualizations, uncertainty plots, GIF demos, and MP4 demos when local codecs are available.

These contributions are software and methodology contributions. Hardware validation and formal safety guarantees are explicitly left for future evaluated work.
