# Code of Conduct

DynNav follows the Contributor Covenant principles: be respectful, constructive, and inclusive.

## Expected behavior

- Use welcoming and professional language.
- Focus technical discussion on evidence, reproducibility, and maintainability.
- Respect different levels of experience.
- Clearly distinguish implemented results from planned work.

## Unacceptable behavior

Harassment, discriminatory language, personal attacks, and knowingly misleading research claims are not acceptable.

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict participation when behavior conflicts with this code of conduct.
