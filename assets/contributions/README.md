# DynNav Contribution Visuals

This directory contains one accessible SVG overview for every contribution from **C01 through C26**. Each figure uses the shared visual language documented in [`assets/styleguide/README.md`](../styleguide/README.md).

The images explain each contribution's input, mechanism, output, or central research idea. They are synthetic explanatory diagrams—not benchmark results, hardware evidence, certified-safety evidence, or formal proofs.

| Range | Themes |
|---|---|
| C01–C07 | planning, uncertainty, risk, recoverability, supervision, resources, exploration |
| C08–C10 | security, multi-robot coordination, human-aware navigation |
| C11–C18 | reinforcement learning, prediction, world models, causal risk, sensing, federation, semantic maps, safety shields |
| C19–C26 | mission language, explanations, policy learning, curriculum, neural scene representations, adversarial testing, swarm consensus |

Files follow the stable naming contract `C##_overview.svg` so documentation and the Streamlit dashboard can reference them deterministically.
