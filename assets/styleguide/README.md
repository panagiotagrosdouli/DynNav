# DynNav Visual Language

DynNav diagrams use a shared dark research-software style so contribution figures remain recognizable and readable across GitHub, documentation, slides, and the Streamlit laboratory.

## Palette

| Role | Hex |
|---|---|
| Canvas | `#020617` |
| Surface | `#0f172a` |
| Border | `#334155` |
| Primary | `#14b8a6` |
| Planning | `#2563eb` |
| Learning | `#7c3aed` |
| Decision | `#ea580c` |
| Main text | `#f8fafc` |
| Secondary text | `#94a3b8` |

## Diagram contract

- Every SVG includes a `<title>` and `<desc>` for accessibility.
- Figures describe synthetic research mechanisms; they are not experimental or hardware evidence.
- Contribution overview figures use four auditable stages: input, mechanism, decision output, and evidence/metrics.
- Text remains selectable and the files contain no external fonts or embedded raster data.
