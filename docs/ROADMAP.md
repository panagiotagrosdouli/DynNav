# Roadmap

## Near term

- Add a geometric-risk baseline with identical scenario seeds.
- Add benchmark aggregation across seed sweeps.
- Add generated planner comparison figures.
- Extend monitoring logs with per-timestep risk and safe-mode transitions.
- Add ROS 2 adapter smoke tests with mocked messages.

## Medium term

- Replace lightweight uncertainty propagation with a calibrated occupancy-belief update.
- Add moving-obstacle scenarios.
- Integrate with Nav2 costmaps and compare against standard planners.
- Add hardware trial manifests and raw logs.

## Long term

- Formalize safety specifications and runtime monitor semantics.
- Validate risk calibration against real sensor data.
- Study multi-robot and semantic exploration extensions.

All roadmap items are future work unless implemented and evaluated in the repository.
