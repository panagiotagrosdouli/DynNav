# C27 — Recoverability Theory for Autonomous Navigation

Recoverability is proposed as a first-class navigation quantity describing how many safe future options remain available to an autonomous robot while operating in unknown environments.

## Core ideas
- Recoverability metric R(x)
- Recoverability Map
- Recoverability Gradient
- Recoverability Horizon
- Recoverability Budget
- Recoverability Barrier
- Recoverability-aware planning

Status: Research proposal and future implementation.
