# Contribution 07 — Next-Best-View με Επίγνωση Επιστροφής

Η Contribution 07 μελετά την ενεργή εξερεύνηση σε άγνωστα περιβάλλοντα. Η επιλογή επόμενης θέσης παρατήρησης δεν βασίζεται μόνο στο αναμενόμενο information gain, αλλά συνυπολογίζει:

- κόστος μετάβασης,
- κίνδυνο διαδρομής,
- δυνατότητα επιστροφής ή ανάκαμψης,
- ποιότητα επικοινωνιακής σύνδεσης.

## Τρέχουσα κατάσταση φακέλων

Υπάρχουν προσωρινά δύο φάκελοι:

- `07_nbv_exploration`: αρχική υλοποίηση frontier/information-gain, την οποία χρησιμοποιούν υπάρχοντα tests.
- `07_next_best_view`: νεότερη returnability-aware επέκταση.

Δεν πρέπει να διαγραφεί κανένας από τους δύο πριν μεταφερθούν ο κώδικας, τα tests και τα αποτελέσματα σε μία ενιαία δομή. Η σχετική διαδικασία περιγράφεται στο [`MIGRATION.md`](MIGRATION.md).

## Επιστημονικός περιορισμός

Το υπάρχον benchmark χρησιμοποιεί συνθετικά candidate viewpoints και χειροκίνητα βάρη. Δεν αποτελεί ακόμη πλήρη επικύρωση σε πραγματικό αισθητήρα ή ρομπότ.
